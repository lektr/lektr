# Quick Start

Get started with Lektr in under 5 minutes.

## 1. Log In

Navigate to your Lektr instance and log in with the default admin credentials:

- **Email**: `admin@lektr.local`
- **Password**: `admin123`

:::warning
Change your password immediately after first login via the user settings.
:::

## 2. Import Your First Highlights

Go to **Import** and choose your source:

### From Kindle

1. Connect your Kindle via USB
2. Navigate to `documents/My Clippings.txt`
3. Upload the file to Lektr

### From KOReader

1. Export highlights from KOReader as JSON
2. Upload the exported file to Lektr

### Manual Entry

Click "Add Highlight" to manually enter a quote with book information.

## 3. Explore Your Library

After import, visit **Your Library** to:

- Browse books and their highlights
- Add tags to organize content
- Edit highlight text and notes
- Pin favorite books

## 4. Search Your Highlights

Use the **Search** feature for semantic search:

- Searches across content, book titles, and authors
- Uses AI-powered hybrid search for best results
- Results ranked by relevance

## 5. Start Reviewing

Go to **Review** to practice spaced repetition:

- Cards are presented based on FSRS algorithm
- Rate how well you remember each highlight
- Build long-term retention
